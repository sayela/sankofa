Sankofa is a clean, tableless, multi-column theme with a wide image slideshow ready for use.

Features

  • Multiple columns - 1, 2, and 3
  • Tableless design
  • Ready to use slideshow
  • Customizable social media integration - Facebook, Twitter, Google Plus, LinkedIn

The slideshow is based on the popular NIVO Slider.

Configuring NIVO Slider

  1. Install the theme by downloading and extracting it to sites/all/themes or sites/sitename/themes
  2. Download the latest version of the Nivo Slider jQuery plugin, extract it somewhere and copy this: jquery.nivo.slider.js to sites/all/libraries or sites/sitename/libraries as you require.
  3. Clear your cache and refresh the page
